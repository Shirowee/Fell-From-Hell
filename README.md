# Fell From Hell

> Projet de L2
Fell From Hell est un jeu de platforme en 2D du genre Beat Them All, dans lequel le joueur affronte des vagues d'ennemis en essayant de survivre le plus longtemps possible au travers des différentes maps. Le projet à été réalisé dans le cadre d'un projet de fin d'année de L2 informatique 0 l'université du Mans.Il a été réaliser avec la bibliothèque Raylib ainsi que cJSON pour traiter des fichiers JSON.

---

Projet réalisé dans le cadre du cours **Gestion de projet**.

## Installation

### Prérequis
- Compilateur C

## Compilation Linux:
Dans fell-from-hell/
```
 make
```
Puis exécuter ***bin/game***

## Compilation Windows WSL:
Dans fell-from-hell/
```
 sudo apt install mingw-w64
 make
```
Puis exécuter ***bin/game.exe***
