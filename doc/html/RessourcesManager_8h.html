◆ RM_GetTexture()

Texture2D * RM_GetTexture ( const char * path)

Récupère une texture depuis le gestionnaire.

Si la texture est déjà chargée, elle est simplement retournée. Sinon, elle est chargée puis stockée.

Parameters
pathChemin vers le fichier image
Returns
Pointeur vers la texture, ou NULL en cas d'erreur
Note
La texture reste en mémoire jusqu'à appel de RM_UnloadTexture() ou RM_UnloadAll()

void RM_UnloadAll ( void )

Décharge toutes les textures.

Libère toutes les textures chargées en mémoire GPU et réinitialise le gestionnaire.

Warning
Doit être appelé avant la fermeture du programme
