Detailed Description

Représente un projectile dans le jeu.

Contient toutes les informations nécessaires pour gérer un projectile : mouvement, dégâts, durée de vie et type de comportement.

Le comportement du projectile dépend du champ indice (BASIC, EXPLOSIF, RAYON, MELEE).
