◆ NewLevel()

int NewLevel ( int newFlag)

Gère l'état de déclenchement d'un nouveau niveau.

Parameters
newFlagIndique si un changement d'état est demandé (1 = toggle, 0 = aucun changement)
Returns
L'état actuel du nouveau niveau (1 = actif, 0 = inactif)

Cette fonction utilise une variable statique pour mémoriser l'état entre les appels. Si newFlag est à 1, l'état est inversé (toggle). Si newFlag est à 0, l'état reste inchangé.

Note
Fonctionne comme un interrupteur (toggle).
