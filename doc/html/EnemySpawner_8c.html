Detailed Description

Gestion de l'apparition dynamique des ennemis.

Author
A. Pocholle

Responsable du spawn automatique des ennemis à intervalles réguliers.

Le système fonctionne avec :

  • un timer interne
  • un cooldown variable
  • un spawn aléatoire sur l'écran
