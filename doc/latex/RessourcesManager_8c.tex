\doxysubsection{Detailed Description}
Gestion des ressources du jeu. 

Définit l’interface de gestion des ressources du jeu, y compris les textures, sons et autres éléments graphiques.

Fournit des fonctions pour \+:
\begin{DoxyItemize}
\item charger les ressources
\item libérer les ressources
\item accéder aux ressources
\end{DoxyItemize}
