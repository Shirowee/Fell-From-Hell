\doxysubsection{Detailed Description}
Gestion du pool d\textquotesingle{}ennemis (object pooling) 

\begin{DoxyAuthor}{Author}
A. Pocholle
\end{DoxyAuthor}
Implémente un système de pool d\textquotesingle{}ennemis basé sur l’object pooling.

Les ennemis sont pré-\/alloués dans un tableau et réutilisés via un flag "{}active"{}, évitant les allocations dynamiques répétées en jeu.

Fonctionnalités \+:
\begin{DoxyItemize}
\item Initialisation du pool
\item Spawn d’ennemis
\item Mise à jour des ennemis
\item Affichage des ennemis
\item Libération mémoire 
\end{DoxyItemize}
