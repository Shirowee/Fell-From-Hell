\doxysubsection{Detailed Description}
Gère le système de spawn des ennemis. 

Permet de contrôler l’apparition des ennemis dans le jeu via un système de timer et de cooldown.

Le timer accumule le temps écoulé, et le cooldown définit le temps minimum entre deux spawns.
