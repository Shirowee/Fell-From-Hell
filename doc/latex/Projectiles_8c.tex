\doxysubsection{Detailed Description}
Gestion des projectiles du jeu. 

\begin{DoxyAuthor}{Authors}
A. Pocholle, Kilian
\end{DoxyAuthor}
Ce module gère la création, la mise à jour et l\textquotesingle{}affichage des projectiles.

Plusieurs types de projectiles sont supportés \+:
\begin{DoxyItemize}
\item BASIC \+: projectile classique
\item EXPLOSIF \+: projectile avec explosion
\item RAYON \+: tir instantané type laser
\item MELEE \+: (non implémenté) 
\end{DoxyItemize}
