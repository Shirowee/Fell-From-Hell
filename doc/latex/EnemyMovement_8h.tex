\doxysubsection{Detailed Description}
Gestion du mouvement des ennemis. 

Contient les fonctions responsables du déplacement des ennemis dans le jeu.

Implémente différentes logiques de mouvement selon le type ou l’état de l’ennemi, par exemple \+:
\begin{DoxyItemize}
\item déplacement aléatoire
\item poursuite du joueur
\item déplacement directionnel simple
\item comportement basé sur une machine à états
\end{DoxyItemize}
