\doxysubsection{Detailed Description}
Représente un pool de projectiles (object pool) 

Cette structure permet de gérer un ensemble de bullets pré-\/allouées afin d\textquotesingle{}éviter les allocations dynamiques répétées pendant l\textquotesingle{}exécution.

Le pool fonctionne en réutilisant les bullets inactives.
