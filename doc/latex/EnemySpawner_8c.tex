\doxysubsection{Detailed Description}
Gestion de l\textquotesingle{}apparition dynamique des ennemis. 

\begin{DoxyAuthor}{Author}
A. Pocholle
\end{DoxyAuthor}
Responsable du spawn automatique des ennemis à intervalles réguliers.

Le système fonctionne avec \+:
\begin{DoxyItemize}
\item un timer interne
\item un cooldown variable
\item un spawn aléatoire sur l\textquotesingle{}écran 
\end{DoxyItemize}
